GoCar

Aplicación de alquiler de vehículos entre particulares

Descripción

GoCar es una aplicación desarrollada en Java que permite el alquiler de vehículos entre usuarios. Incluye gestión de usuarios, reservas y base de datos SQL.

Tecnologías
Java
MySQL
Git
GitHub
Funcionalidades

- Registro de usuarios
- Sistema de reservas
- Gestión de vehículos
- Persistencia en base de datos
